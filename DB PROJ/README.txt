Setup and local host

1. Pip install flask and mysql-connector-python. Versions used in program were:

			 flask: 3.1.0
	mysql-connector-python: 9.3.0

2. Open App.py and change the values in the connector to fit your database made with the database dump on github. It is all the way at the top of the file, under the "CHANGE THESE VALUES" comment.

3. Save and open the terminal in the same folder as the App.py and run the command:

	python App.py

4. Now you've set up a local host server for the website with the IP being in the terminal window, default IP being:

	127.0.0.1:5000
